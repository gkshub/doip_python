### Virtual Environment

1. Create the Environment
> ```bash
> python3.14 -m venv my_env
> ```

2. Activate the Environment
> ```bash
> source my_env/bin/activate
> ```

3. Use and Deactivate
> ```bash
> python main.py
> pip install packages
> deactivate
> ```
